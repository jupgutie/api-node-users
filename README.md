#  API Node Users

##  Descripción
API REST desarrollada con Node.js y Express para la gestión de usuarios.

##  Tecnologías
- Node.js
- Express
- JavaScript

##  Endpoints

### GET /users
Obtiene la lista de usuarios

### POST /users
Crea un nuevo usuario

Ejemplo:
```json
{
  "name": "Juan",
  "age": 25
}

Ejecución
npm install
node index.js
